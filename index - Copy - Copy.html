Tentang Saya

Halo! nama saya Muhammad Hilal Lazuardi Toha. Saya adalah seorang pengembang web yang antusias menciptakan aplikasi web yang fungsional dan menarik. Saya memiliki minat besar dalam desain UI/UX dan pengembangan front end.

Saya selalu berusaha untuk mengikuti tren teknologi terbaru dalam pengembangan web, termasuk teknik responsive design, UI/UX modern, dan framework populer seperti React, Vue, dan Angular.

Dengan pengalaman lebih dari 3 tahun dalam bidang pengembangan web, saya telah menyelesaikan berbagai proyek dari website sederhana hingga aplikasi web kompleks. Saya percaya bahwa desain yang baik harus menyeimbangkan estetika dan fungsionalitas.

Keahlian:

  • HTML5 & CSS3
  • JavaScript (ES6+)
  • React.js
  • Node.js
  • Responsive Web Design
  • UI/UX Design
