Tentang Saya

Halo! nama saya Muhammad Hilal Lazuardi Toha. Saya adalah seorang pengembang web yang antusias menciptakan aplikasi web yang fungsional dan menarik. Saya memiliki minat besar dalam desain UI/UX dan pengembangan front end.

Saya selalu berusaha untuk mengikuti tren teknologi terbaru dalam pengembangan web, termasuk teknik responsive design, UI/UX modern, dan framework populer.

Proyek Karya

Proyek 1: Aplikasi Accord

Aplikasi ini membantu pengguna mencari produk tas yang berkualitas. Dibuat dengan React dan Node.js, menyediakan fitur pencarian yang intuitif dan filter produk yang detail.

Proyek 2: Aplikasi Surat

Aplikasi ini adalah platform pemerintahan untuk melakukan pemilu secara online. Menggunakan teknologi blockchain untuk menjamin keamanan dan transparansi dalam proses pemilihan.

Proyek 3: Portfolio Website

Website portofolio responsif yang dibuat dengan HTML5, CSS3, dan JavaScript. Mengimplementasikan CSS Grid dan Flexbox untuk layout yang dinamis di berbagai ukuran layar.
